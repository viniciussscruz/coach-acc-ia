# Dashboard Frontend (React + Vite + TypeScript)

## Requisitos

- Node.js 20+ (ou 18+)
- npm

## Comandos

```powershell
cd web/dashboard
npm install
npm run build
```

Build final: `web/dashboard/dist`

## Desenvolvimento local

```powershell
cd web/dashboard
npm install
npm run dev
```

Por padrao, o app em dev busca dados de `http://127.0.0.1:8765/api/state`.
